# Adaptive Quiz OCR for GCSE

This repository contains the code for an adaptive quiz system designed for GCSE preparation using OCR techniques. 

## Features
- Adaptive learning paths based on user performance.
- Real-time feedback and performance analytics.
- Integration with existing educational resources.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Vegita936/codig-paper-1-adaptive-quiz-ocr-gcse-.git
   ```
2. Install dependencies:
   ```bash
   cd codig-paper-1-adaptive-quiz-ocr-gcse-
   npm install
   ```

## Usage
- Start the application:
  ```bash
  npm start
  ```
